# How to Setup

```Console
pip install -r requirements.txt
```

# How to Use

## Open3D

```Console
python Open3D/create_pcd.py 
python Open3D/Visualization.py --file_name="Open3D/result.pcd"

or 

python Open3D/equirectangular2spherical_pcd.py
python Open3D/Visualization.py --file_name="Open3D/result.pcd"

or (default data)

python Open3D/Visualization.py
```
